OSMIUM SOUND

A touchscreen-first hi-fi media appliance built on Debian for x86 hardware. Bit-perfect audio, streaming services, and OTA updates — all in one sleek dark interface.

What it does

Everything your hi-fi setup needs

Built for audiophiles who want a dedicated, appliance-grade media server without the complexity.

🎵

High-Resolution Audio

Native playback of FLAC, DSD, and PCM up to 192kHz. Bit-perfect DoP output over USB — no resampling, no compromise.

Integrated access to Deezer, Qobuz, TIDAL, and Spotify — unified in one interface, no app switching.

Release history

Changelog

What's new in each stable release. Development builds ship more often on the Dev OTA channel.

v2.5.13 Latest 30 Jun 2026
  • NewJapanese, Chinese & Korean album and track names now display correctly (CJK fonts bundled).
  • FasterMuch smoother music library — lighter "now playing" updates and fluid scrolling even through thousands of albums.
  • FasterMore responsive Settings: background services no longer block the screen during updates, scans or network changes.
  • FixedHardened the Sources web service and removed dead/duplicate system code.
v2.5.11 26 Jun 2026
  • NewSetup wizard can now install Lyrion Media Server automatically if it's missing.
  • NewUEFI Secure Boot support, applied at image-build time.
v2.5.10 26 Jun 2026
  • NewNative Sources manager — add and manage USB, SMB and local music sources straight from the touchscreen.
  • NewAutomatic USB drive mounting; SMB mounts retry in the background so they survive a fast boot.
  • NewAndroid-style submenu navigation in Settings, with styled in-app confirm dialogs for reboot/shutdown.
  • FasterQuicker boot — slimmer initramfs, masked boot-delaying services, and unused Bluetooth dropped at startup.
v2.5.9 23 Jun 2026
  • NewFull playback controls — shuffle / repeat, play queue, sleep timer, playback preferences and alarms.
  • NewBrowse and save playlists in the Music tab, plus a "Rescan library" button.
  • FixedOS version stays visible when an update check fails; OTA manifest now served from GitHub Pages to dodge API rate limits.
  • FasterLower renderer CPU usage from the analog VU meter, and a smoother boot intro on the x86 mini-PC.
v2.5.8 22 Jun 2026
  • NewRebrand to Osmium Sound, with a brand-new presentation site.
  • NewSigned OS OTA pipeline — update runner, apply.d migrations and idempotency checks in CI.
